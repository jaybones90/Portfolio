Type Test

This web application is a typing test for programmers that tests their keyboarding skills. You can create a new player, select a language to practice, and get your score for characters per minute and which keys you missed. It also uses the D3 Javascript library to generate fun graphics based on correct/incorrect keystrokes.

Angular | D3.js | Firebase | TypeScript | Node | Bootstrap
